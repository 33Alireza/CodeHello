# Contributing to CodeHello 👋

Thank you for considering contributing to CodeHello!  
This project thrives thanks to contributors like you.

## How to Contribute

### 1. Adding a New Language

1. Fork the repository
2. Create a new folder inside languages/ using the language name in kebab-case (e.g. rust, brainfuck, visual-basic)
3. Inside that folder, add:
   - hello.<extension> — the Hello World program
   - README.md — instructions on how to run it

### 2. File Structure Example

```bash
languages/
  └── rust/
      ├── hello.rs
      └── README.md
```

### 3. What Makes a Good Contribution
Use idiomatic code for the language
Keep it simple (classic "Hello, World!")
Include clear run instructions in the folder's README.md
Add the language to the main README if appropriate
Test your code before submitting

## Pull Request Guidelines

- One language per Pull Request (keeps reviews fast)
- Use a clear title: `Add: Rust` or `Fix: Python example`
- Make sure your code runs correctly
- Update the main README only if needed

## Code Style

- Follow common conventions for each language
- Keep "Hello, World!" minimal and clean
- No unnecessary dependencies

## Need Help?

- Open an Issue if you have questions
- Ask for help in your PR description
